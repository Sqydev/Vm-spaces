This project is colletion of scripts that setup and run an virtual workspace for you that you can run fully inside the terminal.<br>
<br>
Warning: This vm setup DOES NOT care about isolationg the vm. So if you run some malware on the vm you're likley to get hacked on your host too.<br>
<br>
To setup:<br>
1. Configure it by edditing config.sh
2. Run createVm.bash
<br>
To run: Just open runVm.bash
